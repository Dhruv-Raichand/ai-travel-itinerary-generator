# MERN AI Itinerary Generator

AI-powered travel itinerary generator built with the MERN stack.

## Features

- JWT Authentication
- AI-generated travel itineraries
- PDF booking parsing
- MongoDB storage
- Shareable itineraries

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- Gemini API

## Backend Setup

```bash
cd server
npm install
npm run dev
````

## Environment Variables

Create a `.env` file inside `server/`

```env
MONGODB_URI=
JWT_SECRET=
GEMINI_API_KEY=
```
